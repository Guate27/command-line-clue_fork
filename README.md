## Solución al misterio

- Con las pistas recopiladas se logró determinar que el responable del homicidio fue el oficial de policía                                  

- El sospechoso confeso que el arma homicida fue una pluma estilográfica

- Gracias a la colaboración de los testigos se conluyó que el lugar donde ocurrió el homicidio fue la cafetería
